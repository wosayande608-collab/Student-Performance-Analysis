# Student Academic Performance Analysis

## Project Overview
This project analyzes a publicly available student performance dataset using Tableau to identify factors that influence academic performance and present insights through interactive dashboards.

## Tools Used
- Tableau
- Git
- GitHub

## Dataset
- Publicly available Student Performance Dataset

## Project Objectives
- Analyze student academic performance.
- Explore the relationship between study time, failures, absences, gender, and school performance.
- Build interactive dashboards.
- Generate insights and recommendations.

## Dashboards
1. Student Academic Performance Analysis
2. Student Performance Insights & Recommendations

## Key Insights
- Higher study time is generally associated with better final grades.
- Students with more academic failures tend to have lower final grades.
- Higher absenteeism is linked to lower academic performance.
- Male students had a slightly higher average final grade in this dataset.
- Academic performance varied slightly between schools.
